Security First

Every skill is built for trust

Unlike other agent marketplaces, LazyClaw enforces security at every layer — from installation to execution.

🔐
AES-256-GCM Encryption
All skill data, configurations, and outputs are encrypted at rest. The server never sees your plaintext.
✓ Unlike OpenClaw's plaintext storage
📦
Sandboxed Execution
Code skills run in isolated Python sandboxes with restricted imports. No skill can access another user's data.
✓ Fully isolated per-user execution
🔍
Security Scanning
Every community skill is scanned for malicious patterns, data exfiltration, and unsafe system calls before listing.
✓ Automated vulnerability detection
🛡️
Permission Model
Three-tier allow/ask/deny permissions per skill. You decide exactly what each skill is allowed to do.
✓ Granular per-tool control
🔑
Encrypted Credential Vault
API keys and secrets are stored in an encrypted vault — never in plaintext .env files that any process can read.
✓ Zero plaintext secrets on disk
👤
Complete User Isolation
Every database query is scoped by user_id. No cross-user data access is architecturally possible.
✓ Hard data boundary enforcement
